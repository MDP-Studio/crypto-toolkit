Calculator guide

Cryptography calculators for AES, RSA, HMAC-SHA1, HMAC-SHA256, SHA, Base64, EC math, and ciphers.

CryptoToolkit is a browser-only educational lab with exact calculators and attack demos. This page lists the high-intent tools searchers usually look for, then links into the interactive modules.

AES and authenticated encryption

AES-GCM shows authenticated encryption, GHASH, nonces, tags, and Web Crypto comparison. AES-128 Round breaks down SubBytes, ShiftRows, MixColumns, and AddRoundKey.

RSA keys and RSA attack math

RSA Key Generator creates browser-side teaching keys and performs encrypt/decrypt checks. Related RSA demos cover factoring, textbook RSA, Wiener's attack, and CRT fault recovery.

HMAC, SHA, and password hashing

HMAC-SHA256 walks through ipad/opad and RFC test vectors. Cipher Tools includes a legacy HMAC-SHA1 document-ID helper for compatibility checks. Base & Encoding includes SHA-1 and SHA-256 hashing. Argon2id demonstrates memory-hard password hashing.

Base64, hex, binary, decimal, and text

Base & Encoding converts text, hex, binary, decimal, and Base64, with LF/CRLF-aware hashing for repeatable learning inputs.

Elliptic curves and signatures

Elliptic Curves includes point addition, scalar multiplication, Montgomery ladder, and baby-step giant-step ECDLP. ECDSA Signing shows hash, sign, verify, and nonce discipline.

Number theory calculators

Modular Arithmetic covers inverse, exponentiation, GCD, extended GCD, Euler's totient, square roots mod p, Legendre symbol, and Miller-Rabin primality. Factorization includes trial division, Pollard's rho, divisor count, and next-prime tools.

Classical ciphers and analysis

Cipher Tools includes Caesar encrypt/decrypt/brute force, Vigenere, ROT13, Atbash, and frequency analysis. Substitution Analysis adds interactive frequency, digraph, and trigraph support.

Protocol and workflow demos

Explore Diffie-Hellman, TLS 1.3, Paillier, ElGamal, Shamir Secret Sharing, and LWE for applied crypto workflows.

What is not included

  • No JWT decoder is listed because CryptoToolkit does not currently ship a JWT calculator.
  • The implementations are educational and client-side. They are not production cryptography libraries.
  • The app is hash-routed, so these search pages link into modules instead of creating fake server routes for each calculator.
